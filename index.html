🚀 Quick Install

        
1
Upload Theme
Zip the theme folder → WP Admin → Appearance → Themes → Add New → Activate

        
2
Install Elementor
Plugins → Add New → Elementor → Install & Activate (free version works)

        
3
Import Template
Elementor → My Templates → Import → upload elementor-template.json

        
4
Insert & Edit
Open any page → Edit with Elementor → Add Template → Insert → Publish
